"DON'T GET SCAMMED" 1만뷰 후킹 카피인데 라이브러리화·시리즈화 안 됨. 잘된 카피 패턴이 반복되지 못함

07 / 17 5FUNNEL 6대 솔루션 ①

진단에 정확히 매칭되는
처방 6가지

SOLUTION 01

카테고리 시그널 시스템화

왜? 알고리즘은 채널 정체성 학습에 시간이 걸립니다. 매번 다른 톤·언어·포맷이면 학습이 리셋됩니다.

요일 × 언어 × 카테고리 매트릭스 강제 (월/영어/매장체험, 화/중번체/시술정보, 금/태국어/트렌드)
1편 = 메인 1언어 + 자막 1~2언어 (6언어 동시 분산 X)
모든 영상 통일 인트로 3초 (강남언니 로고 + 매장 1초 컷)
썸네일 컬러·폰트 통일 (인스타 마케팅 가이드 핵심 원칙 적용)
SOLUTION 02

검증된 글로벌 바이럴 포맷 차용

왜? 강남언니 계정에는 Before/After·POV·리액션 같은 30만+ 뷰 검증 포맷이 한 편도 없습니다.

매월 글로벌 30만+ 뷰 K-Beauty·뷰티 컨시어지 영상 큐레이션
강남언니 톤·자산에 맞는 3~4개 포맷 선정 → 변형 제작
개인 vlog·시딩 형식 X — 자사 채널 운영 포맷만 사용
SOLUTION 03

잘된 패턴 시리즈화 시스템

왜? 강남언니의 1만뷰 콘텐츠는 모두 단발성. 시리즈 누적되면 알고리즘이 "정기 발행 채널"로 인식해 추천 가속.

검증된 매장+컨셉 패턴을 5FUNNEL 포맷에 자연 흡수
시리즈마다 통일 비주얼 코드 (인트로·썸네일·BGM·자막 위치)
매주 같은 요일 = 같은 시리즈, 회차 표기 (시즌 1, 에피소드 12)
SOLUTION 04

후킹 카피 라이브러리

왜? "DON'T GET SCAMMED"가 1만뷰를 만든 이유는 외국인이 가장 두려워하는 단어를 첫 3초에 보여줬기 때문입니다.

외국인 K-Beauty 검색 키워드 + 후킹 카피 50개+ 라이브러리 구축
매주 1개 후킹 카피 → 1편 영상 매핑
카피별 성과 추적 → 다음 달 비중 조정
SOLUTION 05

CTA 동선 표준화

왜? 영상이 100만뷰가 나도 LINE/WeChat 채팅으로 이어지지 않으면 강남언니의 KPI(서비스 알리기)에 기여하지 못합니다.

모든 영상 마지막 3초 = "Free 1:1 consultation, link in bio" 영문 자막
캡션 첫 줄 = "🔗 linktr.ee/UnniGuide"
댓글 자동응답 트리거 키워드 (CONSULT·PRICE·BOOK)
영상별 클릭율 → linktr.ee 분석 연동
SOLUTION 06

매장 시각 자산 다각화

왜? 강남언니 신사옥은 가장 강력한 시각 자산. 그러나 같은 앵글이 6편+ 반복되면 알고리즘이 동일 콘텐츠로 인식해 노출 분산.

매장 외관 = 5개 앵글 정의 (정면·측면·롱샷·디테일·낮/밤)
인테리어 = 4개 공간 (대기실·시연실·상담실·복도)
시연실 3D 피부 분석 = 시리즈 단독 라인
매월 신규 앵글 1개 추가 (계절·이벤트·신규 인테리어)

11 / 17 왜 숏폼인가

언니가이드가 숏폼에 최적인
3가지 구조적 이유

REASON 1

시각적 변화가 강력합니다

피부 시술 비포애프터는 숏폼에서 가장 강력한 스크롤 정지 효과(Stop Effect)를 가집니다. 1초 컷 전환 영상은 검증된 글로벌 K-Beauty 포맷이고, 강남언니는 외국인 모델 풀 + 신사옥으로 안정 제작 가능합니다.

REASON 2

무료 = 진입 장벽 제로

대부분 의료관광 광고는 "그래도 비싸겠지", "외국인이라 호구잡힐 거야" 진입 장벽에 막힙니다.

언니가이드는 상담·통역·수수료 모두 무료, 한국인 동일 가격. 영상에서 약속을 보여주면 진입 장벽 자체가 사라집니다.

REASON 3

전환이 단순합니다

복잡한 의료 서비스는 영상 → 전화 → 상담 → 방문 단계가 많아 이탈이 큼.

언니가이드는 영상 → LINE/WeChat 링크 → 1:1 채팅 3단계. 시간대 차이도, 한국어도 필요 없습니다.
